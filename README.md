## ✨ About The Project

**Dream Booking** is a fictional platform designed to help users find and book luxury resorts around the world.

This project focuses on:

* Clean UI design
* Responsive layouts
* Reusable Tailwind components
* Smooth interactions and animations

## 🎨 Design System

This project uses a **two-level color system**:

### Level 1 — Palette

Raw colors (teal, sand…)

### Level 2 — Semantic Tokens

Used directly in UI:

* `bg-base` → main background
* `bg-alt` → section background
* `surface` → cards
* `accent` → buttons / highlights
* `text-primary` → headings
* `text-body` → content

📌 Defined in: tailwind.config.js

## 📱 Mobile Navigation

* Built using `:focus-within`
* Includes overlay background
* Smooth width + opacity transition
* No JavaScript required

---

## 🎬 Animations

Custom animations defined in Tailwind config:

* `fade-up` → appear + move up
* `fade-in` → simple opacity
* `slide-in` → slide from right
* `jump` → looping animation
